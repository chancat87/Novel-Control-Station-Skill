# Scenario 02: Continuation With Existing Files

## Prompt

"Continue chapter 27. Use the current files. The heroine must push the smuggling line forward, the rival line cannot disappear, and the confession debt from chapter 24 still matters."

## Expected Baseline Failure Without The Skill

- drafts from loose memory
- ignores at least one active line
- forgets prior emotional debt
- uses style loosely without controlled loading

## Required Skill Behavior

- read the required project files before drafting
- read only the relevant style modules for the chapter
- generate a chapter control card
- identify line obligations, emotional debt, style goals, and risks
- draft from the control card rather than improvising freely

## Pass Criteria

- chapter control card exists before drafting
- multiple lines remain active
- prior emotional debt affects the chapter
- style loading is selective rather than global
