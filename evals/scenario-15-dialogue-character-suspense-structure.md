# Scenario 15: Dialogue Character Suspense Structure

## Prompt

"Write a chapter where the heroine's controlling personality shows through the dialogue, a hidden clue changes the power balance, and the chapter should end on a necessary next move rather than a random cliff."

## Expected Baseline Failure Without The Skill

- writes generic dialogue
- treats character as surface label
- uses suspense as withholding without fair pressure
- produces a chapter with events but weak architecture

## Required Skill Behavior

- apply the new execution-method references during planning and review
- make core personality visible in behavior and speech
- make suspense fair, pressured, and consequential
- give the chapter an entry, turn, local closure, and carryover debt

## Pass Criteria

- chapter planning visibly uses character, dialogue, suspense, and structure rules
- the resulting chapter would pass those execution-method checks
- the ending creates necessity rather than empty noise
