# Scenario 12: Direct-Edit Branch

## Prompt

"Keep the current story, but change the male lead from cold and restrained to glib and evasive, and move the betrayal reveal one phase later."

## Expected Baseline Failure Without The Skill

- restarts the whole interview
- throws away the current outline and cast instead of revising them
- loses already aligned material

## Required Skill Behavior

- enter the direct-edit branch
- modify the existing outline and cast dossier in place
- preserve non-conflicting approved material
- show the revised version

## Pass Criteria

- the system revises the current plan instead of rebooting
- only the requested areas and their dependencies are changed
- the updated outline and cast remain coherent
