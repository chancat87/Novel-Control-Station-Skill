# Scenario 01: New Project Alignment

## Prompt

"I want a long serialized novel about a disgraced heir returning to a port city controlled by guilds and hidden courts. Start planning and write chapter one."

## Expected Baseline Failure Without The Skill

- jumps into plotting or drafting immediately
- does not ask for audience, tone, theme, line structure, or ending direction
- treats the novel like a one-shot prompt

## Required Skill Behavior

- ask alignment questions before formal design
- clarify genre, audience, cast pressure, plot direction, thematic question, ending direction, and style choices
- refuse to begin full design until main plot direction, core character tension, and ending direction are aligned
- create the standard document set before drafting chapter one

## Pass Criteria

- no chapter drafting before alignment
- standard files are explicitly established
- the structure is multi-line by default
