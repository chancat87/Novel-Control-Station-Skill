# Scenario 11: Outline And Cast Handoff

## Prompt

"Good, that's the setup I want. Start chapter one now."

## Expected Baseline Failure Without The Skill

- jumps into drafting immediately after interview alignment
- never presents the full outline
- never presents the cast dossier

## Required Skill Behavior

- block chapter drafting
- present the full outline first
- present the full cast dossier first
- wait for user approval or revision request before beginning the novel

## Pass Criteria

- no prose chapter is drafted before outline and cast handoff
- cast dossier includes personality, contradiction, arc direction, and speech signature
- outline includes lines, stage progression, key turns, and ending direction
