# Literary Module

Literary intensity comes from pressure between language, character, and theme. Depth should arise through scene, image, and choice rather than direct sermon.

## Use When

- thematic pressure needs reinforcement
- image or motif return should carry meaning
- language discipline or formal intensity matters

## Core Effects

- thematic pressure
- image and motif return
- precise language
- psychological depth
- meaningful structural movement
- layered interpretation

## Chapter Use

- define what thematic pressure the chapter applies
- know which motif or image should recur, if any
- use structural complexity only when it reveals character truth or thematic relation

## Revision Checks

- does the chapter press a real thematic conflict
- do images carry weight rather than decoration
- is the language alive instead of inflated
- does structure reveal character truth

## Common Failures

- fake depth through abstraction
- dead language
- nonlinear movement without purpose
- commentary replacing drama
