# Humor Core

In fiction, humor works best when it emerges from pressure.

Useful engines:

- a character trying too hard to seem controlled
- a mismatch between self-image and reality
- status inversion
- repeated avoidance that becomes impossible
- overconfident interpretation meeting hard fact

Prefer:

- character-specific humor
- relational humor
- scene-based embarrassment
- compressed observation

Avoid:

- unrelated one-liners
- funny prose that erases scene logic
- humor that destroys the novel's emotional continuity
