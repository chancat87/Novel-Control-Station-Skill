# Callback And Escalation

Callbacks work when a prior beat returns with changed pressure.

Strong callback pattern:

1. plant a small comic fact
2. repeat it in a harder context
3. transform it into a payoff that reveals character or relationship truth

Escalation pattern:

1. normal awkwardness
2. public complication
3. status damage
4. emotional or plot consequence

Do not escalate only in noise. Escalate in cost.
